# Fiction Prose Fill Guide

How to fill visual and asset stubs when the ship target is prose, not designed campaigns.

Applies to `fiction` and prose-first `ttrpg` worlds. See [`fiction-ttrpg-tracks.md`](fiction-ttrpg-tracks.md).

---

## `22_visual_core` (prose mapping)

| Stub field | Prose equivalent | Example |
| --- | --- | --- |
| Display / Body type | Narrative register | "Short sentences for action; longer for institution scenes" |
| Chromatic palette | Mood palette in words | "Soot gray institutions, rust accent for danger, no pastoral green" |
| Grid | Chapter/scene structure | "Institution scenes: numbered blocks; intimacy scenes: single column prose" |
| Identity marks | Recurring visual motifs in text | "Rectangular seals on letters; round objects only at hearth scenes" |
| Silhouette rule | Shape language in description | "Architecture rectangular; care objects may curve" |
| Figma link | `n/a` | Reference map sketch path if exists |

Do not invent hex codes unless a cover designer needs them. If hex exists, tie each color to a prose mood note.

---

## `23_asset_behavior` (scene texture)

| Stub field | Prose equivalent | Example |
| --- | --- | --- |
| Camera protocol | POV and distance | "Close third for care scenes; wide for institution set pieces" |
| Motion physics | Pacing of cuts | "Scene breaks on decision, not cliffhanger noise" |
| Texture ledger | Sensory palette | "Paper, cold metal, wet wool; ban perfume-commercial scent words" |
| Diegetic vs non-diegetic | In-world vs authorial | "No author winking; epigraphs OK as non-diegetic frame" |

Skip entire doc (`status: skipped`) if the world is manuscript-only with no cover/film.

---

## When to skip vs fill

| Doc | Fill when | Skip when |
| --- | --- | --- |
| `22` | Cover art, maps, illustrated edition | Query + unillustrated MS only (still recommend mood palette) |
| `23` | Film option, illustrated scenes | Pure text, no visual production |
| `31` | Scored AV | Print/tabletop silent |

Record skip in governance Skip Registry with revisit trigger.
