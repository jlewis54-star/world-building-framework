# Notes scratch

Intake landing zone for research captures and meeting notes. **Not canon.**

## Use

1. Save raw captures here (or run `low-ambiguity-notes` and paste output).
2. Promote selected claims into `world/` per [`playbook/notes-to-canon.md`](../../playbook/notes-to-canon.md).
3. Delete or archive scratch files after promotion.

Do not treat this folder as a second bible. Do not copy wholesale into `world/`.

Obsidian vaults and private branches remain valid alternatives. This path is the default in-repo scratch location.
