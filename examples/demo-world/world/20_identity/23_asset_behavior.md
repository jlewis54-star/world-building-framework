# 23 Asset Behavior

## Camera Protocol

| Spec | Rule |
| --- | --- |
| Lenses | 35 mm and 50 mm prime only for doc photography |
| Framing | Eye-level, flat forms; no drone hero shots |
| Grade / grain | Neutral grade; grain allowed on 16 mm only |
| Forbidden | Teal-orange grade, lens flare as decoration |

## Motion Physics

| Context | Speed / easing | Notes |
| --- | --- | --- |
| UI / motion graphics | 200 ms linear | No bounce |
| Editorial cuts | Hard cuts; 12 fps for archive inserts | |
| Forbidden motion | Parallax on form fields | |

## Texture Ledger

| Texture | Where used | Banned adjacent |
| --- | --- | --- |
| Uncoated paper | Print, tags | Gloss laminate |
| Cold-rolled steel tags | Object editions | Painted faux rust |
| Vinyl desk matte | Photo backgrounds | Marble, wood grain |

## Diegetic vs Non-Diegetic

| Class | Allowed | Forbidden |
| --- | --- | --- |
| Diegetic | Stamps, manifests, radio static, pier PA | Floating UI cards in-world |
| Non-diegetic | Episode titles, captions, archive headers | Fake handwritten overlays on digital UI |
