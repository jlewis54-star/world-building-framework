# 21 Verbal Lexicon

## Voice Profile

| Attribute | What it means | What it is not |
| --- | --- | --- |
| Clerk-precise | Names forms, dates, pier numbers | Cold legalese |
| Radio-warm | Short sentences, spoken cadence | Podcast hype |
| Material-honest | Describes ink, steel, paper weight | Craft marketing fluff |

## The Dictionary

| Term | Definition | Usage note |
| --- | --- | --- |
| Double manifest | Shipment listed twice with conflicting dates | Use sparingly; never as joke |
| Tag run | Numbered object edition tied to a radio episode | Always include run size |
| Pier ledger | Internal research doc; not public brand name | External: "archive note" |

## Banned Words

| Banned | Why | Use instead |
| --- | --- | --- |
| journey | Generic brand speak | route, shift, episode |
| curated | Implies taste without rules | selected, logged, stamped |
| exclusive | Hype | numbered, limited run |

## Practical Copy Examples

| Context | Wrong | Right |
| --- | --- | --- |
| Social caption | "Embark on a curated maritime journey." | "Tag run 04 ships Monday. 120 steel tags, black ink." |
| Product label | "Premium exclusive experience." | "Pier 9 dispatch form, reprint, 500 copies." |
| Error message | "Oops! Something went wrong." | "Manifest mismatch. Retry or call pier desk." |
| Press blurb | "Revolutionary ocean lifestyle brand." | "Harbor Line documents freight folklore in print and radio." |
