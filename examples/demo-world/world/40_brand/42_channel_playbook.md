# 42 Channel Playbook

## Digital Product (Web / App)

| Spec | Rule |
| --- | --- |
| UI metaphor | Terminal forms, not cards |
| Components banned | Skeleton loaders, glass panels |
| Motion | cite `23` |
| Copy | cite `21` |

## Social / Short-form

| Spec | Rule |
| --- | --- |
| Aspect ratios | 4:5 stills primary |
| Caption voice | cite `21` |
| Hook patterns allowed | Run number, pier reference, date stamp |
| Forbidden formats | Trend audio, green-screen docks |

## Physical Artifacts (Merch / Pack)

| Spec | Rule |
| --- | --- |
| Blanks / materials | cite `41` |
| Label copy | Manifest line + run size |
| Unboxing / ephemera | Include duplicate form insert on tag runs |

## DSP / Streaming / Profile (if music or podcast)

| Spec | Rule |
| --- | --- |
| Avatar / header | HL monogram on warm white |
| Bio | One sentence from public positioning |

## Spatial / Events

| Spec | Rule |
| --- | --- |
| Lighting | Low warm practicals, no gobo logos |
| Materials | Steel tables, paper stacks |
| Scent / sound (optional) | Ink and cold metal only if used |

## Channel Priority

1. Print / object editions
2. Low-power radio archive
3. Social stills (4:5)
