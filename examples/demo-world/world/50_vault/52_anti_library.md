# 52 Anti-Library

## Adjacent Worlds / Competitors

| Name | Why adjacent | Hard ban for us |
| --- | --- | --- |
| Nautical lifestyle brands | Ocean imagery | Rope textures, anchor icons |
| Supply-chain SaaS marketing | Logistics vocabulary | Dashboard card UI |
| True-crime podcast aesthetics | Document tone | Sensational copy |

## Aesthetic Traps

| Trap | Why seductive | Rejection rule |
| --- | --- | --- |
| Faux patina | Reads "authentic" fast | Real wear only, documented |
| Teal-orange grade | Popular in maritime B-roll | Neutral grade only |
| Handwritten overlay fonts | Implies personal myth | Mono forms for institutions |

## Language Traps

| Phrase / mode | Rejection rule |
| --- | --- |
| "Set sail on your journey" | Delete; use route/shift language |
| "Exclusive drop" | Use numbered run + date |
| "Crafted with love" | State material and run size |
